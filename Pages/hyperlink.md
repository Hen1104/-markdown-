# Hyperlink
How To Make Link Like `<a href>`?

## Getting Started
Basic

```md
[Link](http://url.com/)
```
Output :
[Link](http://url.com/)


Added **Font Style**

```md
// This is bold text Hyperlink //
**[Link](http://url.com/)**
```
Output: **[Link](http://url.com/)**
